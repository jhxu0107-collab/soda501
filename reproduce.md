# Reproducibility

This repository contains a fully reproducible analysis of the relationship
between education and income. All main results were generated using Python.

# Environment

Information about the computational environment can be found in
`outputs/session_info.txt`.

# Files

The file `analysis_log.txt`, located in the root directory of the repository,
records key information about the analysis, including:

- the number of rows and columns in the main dataset
- the date and time when the analysis was run
- which output files were generated
